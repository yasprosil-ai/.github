# Yasprosil AI

**Создаю цифровые продукты с помощью AI и вайбкодинга**

Сайты, лендинги, Telegram-боты и MVP — от идеи до первой рабочей версии.  
Быстро. Без лишнего. С фокусом на результат.

---

### Что делаю

- **Сайты и лендинги** — многостраничные сайты, промо-страницы, сайты экспертов
- **Telegram-боты** — запись, FAQ, сбор лидов, воронки
- **Мини-аппы** — калькуляторы, квизы, сборщики брифов
- **MVP** — первая рабочая версия продукта для проверки гипотезы
